# Framework Responsibilities

The framework is responsible for repeatable execution, validation, and handover-ready outputs.

It covers:
- Notebook setup patterns
- Profiling utilities
- Source metadata logging
- Schema drift checks
- Data drift checks
- Incremental safety checks
- Technical columns
- Write pattern
- Output profiling
- DQ execution
- Contract validation
- Lineage metadata capture
- Run summary and AI context export
