# Fabric Data Product Framework

Build production-friendly data products in Microsoft Fabric with reusable notebook patterns, contract-first controls, and handover-ready artifacts.

## Why this framework

- **Contract-first delivery** for safer schema and policy changes.
- **Built-in quality and drift checks** for dependable runs.
- **Lineage and summary outputs** to make AI-assisted handover easier.
- **Fabric-native notebook utilities** for practical implementation speed.
