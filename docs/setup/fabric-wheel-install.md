# Install in Fabric

This guide has been split into two focused pages:

- [Create Wheel](create-wheel.md): build the local `.whl` package.
- [Run in Fabric](run-in-fabric.md): upload the wheel to a Fabric Environment, attach it to notebooks, and verify it runs.

If you are setting up FabricOps for the first time, start with Run in Fabric after creating the wheel.
